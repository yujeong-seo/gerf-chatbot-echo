# ECHO: Event Conversation, Heard, Ongoing

### Master's Project: Event Chatbot for the Great Exhibition Road Festival (GERF)**
* The repository contains replicable SQL schema for Supabase setup.
* Knowledge base (`/rag`) contains PDF based Frequently Asked Questions document, and SQL based event database. Event catalog can be managed and downloaded using `/supabase/schema.sql`.
* Implemented using Supabase, Railway, and Vercel.

### Functionality
* **🔍 Onboarding:** Light segmentation and varying entry options lower cognitive load. Visitors are greeted with an optional name prompt and a selection of broad interest categories for guided step into self-disclosure.
* **👥 Community tab:** Keyword based insights ensure a topical transition from between the collective crowd and the individual chat.
* **🗣️ Personalised chat:** Chain-based routing logic capabilities using agent prompting and ReAct logic to maintain context. Routing logic classifies user intent across four categories for subsequent mechanism query.
* **📲 Function calling:** Integrated inline UI blocks balance text output with actionable, visual cards.
* **💬 Conversational experience:** Agent provides an immediate sense of responsiveness by reflecting user language back in its replied, signalling that the input is registered.
